# Personal Portfolio Website

This project is a personal portfolio website developed for my friend using React.js and Vite. The website showcases skills, projects, certifications, achievements, and contact information through a modern and responsive user interface.

## Features
- Responsive Design
- Modern UI
- Skills Section
- Projects Showcase
- Certifications Section
- Contact Information

## Technologies Used
- React.js
- Vite
- JavaScript
- HTML5
- CSS3
